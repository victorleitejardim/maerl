bg_pcaR Documentation

bg_pca

Description

This function is built on pca output of vegan rda function and uses Aurelien Boye's code as a basis for creating pretty PCAs but with a lot of new cosmetic options

Usage

bg_pca(
  pca,
  axes = c(1, 2),
  metadata = NULL,
  site.scaling = 1,
  species.scaling = 2,
  goodness.axis = 2,
  goodness.thresh = 0.3,
  main.group = NULL,
  second.group = NULL,
  nudge.x = NULL,
  nudge.y = NULL,
  scale.fill = NULL,
  scale.shape = NULL,
  scale.colour = NULL,
  labels = NULL,
  print.sp.thresh = FALSE,
  ext.plot.scale = 5,
  add.stat1 = TRUE,
  stat1 = "chull",
  add.stat2 = FALSE,
  stat2 = NULL,
  add.dens = TRUE,
  add.centroids = FALSE,
  point.size = 2.5,
  c.size = 5,
  p.alpha = 1,
  font.size = 18/.pt,
  axis.size = 22,
  axis.text = 20,
  conf.level = 0.95,
  ysites = NULL,
  xsites = NULL,
  ysp = NULL,
  xsp = NULL,
  add.labels = TRUE
)

Arguments

pca

a vegan pca output

axes

numeric vector;the selected pca axes to plot

metadata

a table containing the metadata mandatory to define the main and second groups

site.scaling

numeric; scaling for the first plot - defaut is scaling 1

species.scaling

numeric; scaling for the second plot - defaut is scaling 2

goodness.axis

numeric; the number of axis used to calculate the goodness of fit of the species

goodness.thresh

numeric; goodness of fit threshold for selecting species

main.group

factor; defines the grouping variable used to calculate the densities and to define the color and fill aesthethics, it cannot be empty

second.group

factor; defines a second level of grouping that, if present, will be highlighted with different shape

nudge.x

numeric vector; defines the gap between the labels and the group centroid in the x axis, it must have the same length as the grouping variables defined in main.group

nudge.y

numeric vector; defines the gap between the labels and the group centroid in the y axis, it must have the same length as the grouping variables defined in main.group

scale.fill

string; contains the palette for fill aesthetics

scale.shape

numeric vector; contains the shape types for shape aesthetics

scale.colour

string; contains the palette for colour aesthetics

labels

factor; factor in metadata containing the label of each level of the grouping variables in main.group or second.group

print.sp.thresh

logical; print the goodness of fit threshold for selecting species' vectors

ext.plot.scale

numeric; scale for density plots in proportion to other plots

add.stat1

logical; add either a convex hull or an ellipse to each group on the first plot

stat1

character; choose between a convex hull "chull" or an ellipse "ellipse" for the first plot

add.stat2

logical; add either a convex hull or an ellipse to each group on the second plot

stat2

character; choose between a convex hull "chull" or an ellipse "ellipse" for the second plot

add.dens

logical; add the density plots

add.centroids

logical; add centroids for each group

point.size

numeric; define sites' point size

c.size

numeric; define centroids' point size

p.alpha

numeric; define the alpha for sites' points

font.size

character; defines the font size for geom_label using the syntax "18/.pt"

axis.size

numeric; defines the font size for the axes titles

axis.text

numeric; defines the font size fot the axes text

conf.level

numeric; defines the C.I. of the ellipse

ysites

numeric vector; force y axis limits for the first plot

xsites

numeric vector; force x axis limits for the first plot

ysp

numeric vector; force y axis limits for the second plot

xsp

numeric vector; force x axis limits for the second plot

add.labels

logical; add geom_label for plotting the group names

Value

returns a grid.arrange with two plots with the species and sites scaling biplots.

Examples

set.seed(100)
